# 🌙 Drift and Flow — A Sleep Game

An interactive sleep experience. Not a game you win — a game you fall asleep to.

## ✨ Features

- 🪼 **9 Bioluminescent Jellyfish** — glowing, pulsing, drifting upward with trailing tentacles
- 🌀 **Breathing Orb** — guides your breathing, gradually slowing over time
- ✨ **Drifting Particles** — soft lights that follow your touch
- 🎧 **Immersive Audio** — layered drones, ocean waves, singing bowls, chimes, and delta-wave binaural beats
- 🌊 **Touch Ripples** — water rings expand where you touch
- 🌑 **Auto Sleep Mode** — world fades into darkness when you stop interacting
- 💤 **No score. No timer. No failure.** Just drifting.

## 🎮 How to Play

1. Open in any browser or visit the GitHub Pages link
2. **Turn your volume up** 🔊
3. Tap to begin
4. Move slowly... breathe... let go...

🎧 Headphones recommended for binaural beats

## 🛠 Tech

Pure HTML5 / Canvas / Web Audio API — zero dependencies, works offline.
